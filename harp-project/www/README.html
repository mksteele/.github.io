Pretty Harp & Jade Skeleton Template

An incredibly simple skeleton template for Harp using the Jade / Pug templating language to get you started. It's not meant to be a perfectly complete blog theme but rather a starting point if you want to experiment with static website generators and build your own template.

This template was originally developed for The Ultimate Guide to Static Websites with Harp & Jade.
